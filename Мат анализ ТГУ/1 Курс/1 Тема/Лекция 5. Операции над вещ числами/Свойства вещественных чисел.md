Свойства сложения:
1. $a+b=b+a$
2. $(a+b)+c = a+(b+c)$
3. $a+0=a$
Доказательство всех этих свойств основывается на теореме о приближении вещественного числа рациональными и леммах [[Лемма о промежуточном числе|1]] и [[Лемма о равенстве чисел|2]].
4. Для каждого вещественного числа a существует противоположное (симметричное) ему число $-a$ , удовлетворяющее условию: $a + (-a) = 0$.
Свойство вычитания:
1. $-(a+b) = (-a)+(-b)$
Свойства умножения:
1. $a*b=b*a$ (переместительное свойство умножения);
2. $(a*b)*c=a*(b*c)$ (сочетательное свойство умножения);
3. существует число 1, обладающее свойством: $a*1=a$;
4. для каждого числа a существует число $\frac{1}{a}$ (обратное), такое, что $a*\frac{1}{a}=1$
5. $(a+b)*c=ac+bc$ (распределительное свойство умножения). (Отсюда легко получить распределительное свойство относительно разности: $(a-b)*c=ac-bc$).
6. Если $a>b$ и $c>0$, то $ac>bc$.
